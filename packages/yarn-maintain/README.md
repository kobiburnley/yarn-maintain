# yarn-maintain

Maintain `yarn.lock`

### Usage

```
npx yarn-maintain [flags]
```

### Options:

| Option | Description |
| --- | ----------- |
| `-m, --module, --modules` | Comma seperated list of module names |
| `-f, --filter, --filter` | Comma seperated list of regex patterns |
| `-s, --scope, --scopes` | Comma seperated list of package scopes |
